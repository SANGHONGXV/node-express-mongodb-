

## 1.部署

1.1 安装node环境

1.2 删除项目根目录下./node_modules文件夹，项目根目录下命令行操作：

> npm install

开发环境执行
> npm run server

或者

生产环境执行
> npm run start

1.3 浏览器访问 http://127.0.0.1:5000
